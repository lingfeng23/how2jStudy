- throws与throw这两个关键字接近，不过意义不一样，有如下区别：
    * throws 出现在方法声明上，而throw通常都出现在方法体内。
    * throws 表示出现异常的一种可能性，并不一定会发·生这些异常；throw则是抛出了异常，执行throw则一定抛出了某个异常对象。
